# photoEdit

# 模仿水印相机界面，对上传的图片进行日期、时间、星期、地点的编辑并下载。

## 可一次上传 4 张图片批量添加水印，上传图片后自动处理并保存

## 打开页面后，选择日期即可，根据日期自动匹配周几。选择好日期后添加图片即可。
